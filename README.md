### [LUMEN](https://github.com/pixelsncodes/lumen)

A C++20 and JUCE 8 wavetable synthesizer delivered as a VST3 plugin and Windows standalone application.

LUMEN converts image data into playable wavetables and includes synthesis, DSP, MIDI, modulation, presets, rendering tools, and automated test infrastructure.

**Stack:** C++20, JUCE 8, CMake, DSP, FFT, MIDI, VST3

### [Lumena](https://github.com/pixelsncodes/lumena)

A standalone image-to-MIDI system that transforms visual information into structured musical output.

It includes image analysis, music-theory-based key selection, deterministic phrasing, Markov-chain melody generation, and a dependency-free MIDI writer.

**Stack:** C++20, CMake, MIDI, generative algorithms, Markov chains
